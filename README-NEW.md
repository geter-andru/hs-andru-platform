# H&S Revenue Intelligence Platform

Enterprise-grade Revenue Intelligence Operating System with advanced ICP analysis, cost calculation, and business case building tools.

## 🚀 Live Platform
**URL**: https://platform.andru-ai.com

## 🛠️ Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🏗️ Tech Stack
- **React** 18.2.0 with Create React App
- **Airtable** for data storage and customer management
- **Make.com** for workflow automation
- **Tailwind CSS** for styling
- **Framer Motion** for animations

## 🎯 Core Features

### Revenue Intelligence Tools
- **ICP Analysis**: Advanced customer profiling with 95+ scoring criteria
- **Cost Calculator**: Financial impact assessment with ROI modeling  
- **Business Case Builder**: Comprehensive proposal generation

### Enterprise Dashboard
- **Modern SaaS Interface**: Professional sidebar navigation, 120px progress charts
- **Mobile-First Design**: Touch optimization, overlay navigation
- **Competency Tracking**: 6-level advancement system with real-world action tracking
- **AI-Powered Resources**: Sales Sage resource generation with Make.com integration
